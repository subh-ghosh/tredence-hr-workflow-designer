# HR Workflow Designer Case Study

This project is a small workflow builder for HR teams.

## What this app does
- Lets HR add steps on a canvas
- Lets HR connect steps in order
- Lets HR edit each step in a side form
- Lets HR run a mock simulation and view step logs

## Tech
- React + TypeScript + Vite
- React Flow
- Local mock APIs

## Main node types
- Start
- Task
- Approval
- Automated
- End

## Run (planned)
- Install packages: `npm install`
- Start dev server: `npm run dev`

## Notes
- This case study is frontend-first.
- No auth and no DB persistence are required.
